1. docker --version
   → Displays the installed Docker version.

2. docker info
   → Shows detailed information about the Docker environment.

3. docker pull <image>
   → Downloads a Docker image from a Docker registry (such as Docker Hub).

4. docker images
   → Lists all Docker images available on the local system.

5. docker run <image>
   → Creates and starts a new Docker container from a specified image.

6. docker run -it <image>
   → Starts a Docker container in interactive terminal mode, allowing users to run commands inside the container.

7. docker ps
   → Lists currently running Docker containers.

8. docker ps -a
   → Lists all Docker containers, including stopped containers.

9. docker start <container>
   → Starts a stopped Docker container.

10. docker stop <container>
   → Stops a running Docker container.

11. docker restart <container>
   → Restarts a Docker container.

12. docker rm <container>
   → Removes a Docker container.

13. docker rmi <image>
   → Removes a Docker image from the local system.

14. docker exec -it <container> bash
   → Opens an interactive bash shell inside a running container.

15. docker logs <container>
   → Displays the logs generated by a Docker container.

16. docker inspect <container/image>
   → Shows detailed information about a Docker container or image.

17. docker build -t <name> .
   → Builds a new Docker image using instructions from a Dockerfile.

18. docker tag <image> <repository>
   → Creates a new tag for an existing Docker image.

19. docker push <image>
   → Uploads a Docker image to a Docker registry.

20. docker login
   → Logs into a Docker registry account.

21. docker logout
   → Logs out from a Docker registry account.

22. docker cp <source> <container:path>
   → Copies files between the host machine and a Docker container.

23. docker volume ls
   → Lists all Docker volumes.

24. docker volume create <name>
   → Creates a new Docker volume for persistent data storage.

25. docker network ls
   → Lists all Docker networks.

26. docker system prune
   → Removes unused Docker containers, images, networks, and other resources.

27. docker stats
   → Displays real-time CPU, memory, and network usage of running containers.

28. docker compose up
   → Starts and runs services defined in a Docker Compose file.

29. docker compose down
   → Stops and removes Docker Compose services.

30. docker create <image>
   → Creates a new container without starting it.

31. docker rename <old> <new>
   → Renames an existing Docker container.

32. docker pause <container>
   → Pauses all processes running inside a container.

33. docker unpause <container>
   → Resumes a paused Docker container.

34. docker top <container>
   → Displays running processes inside a container.

35. docker diff <container>
   → Shows changes made to files inside a container compared to its original image.

36. docker history <image>
   → Shows the history and layers of a Docker image.

37. docker save <image> -o file.tar
   → Saves a Docker image as a tar archive file.

38. docker load -i file.tar
   → Loads a Docker image from a tar archive file.

39. docker commit <container> <image>
   → Creates a new Docker image from a container's current state.

40. docker version
   → Displays Docker client and server version information.
